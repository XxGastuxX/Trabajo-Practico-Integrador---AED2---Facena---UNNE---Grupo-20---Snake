*Proyecto Integrador – Snake*

Este proyecto implementa el clásico juego de la Viborita utilizando el lenguaje C y las estructuras de datos y tecnicas vistas durante la cursada.

El programa incluye:
 ■Listas enlazadas
 ■Colas
 ■Árbol binario de búsqueda
 ■TAD
 ■Persistencia en archivos
 ■Interfaz con menús en ASCII art

*Descripción*

Al ejecutar el juego, el usuario accede a un menú principal donde puede:

■Iniciar una nueva partida

■Ver el ranking de puntajes guardados

■Salir del programa

■Durante la partida, la serpiente podrá moverse por el mapa, comer comida para crecer y sumar puntos, y el juego finalizará si colisiona con los bordes o con su propio cuerpo.

*Todos los puntajes se guardan en un archivo binario (scores.dat) y luego se muestran utilizando un árbol binario ordenado por puntaje.*

*Ejecutar el juego*
Para jugar:

■Abra la carpeta del proyecto.

■Ubique el archivo Snake.exe.

■Ejecútelo con doble click, siempre desde la misma carpeta donde está los archivos .txt y donde estará el archivo de puntajes.
